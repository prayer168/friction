# 摩擦力搜查隊

適合國小高年級的摩擦力互動數位教材。學生會先預測再操作推箱模型，記錄控制變因的測試結果，逐格觀察靜摩擦轉為動摩擦，並以八個生活案例與十題闖關整理證據。

## 學習重點

- 摩擦力反抗接觸面的相對滑動或滑動趨勢。
- 靜摩擦力會在上限內配合外力改變；沒移動不代表沒有摩擦力。
- 簡化模型中，材料組合與正向力會影響摩擦。
- 摩擦力可提供抓地與煞車，也會造成阻力、熱與磨耗。

## 本機使用

直接開啟 `index.html`，或執行：

```powershell
npm.cmd run serve
```

## 驗證

```powershell
npm.cmd test
npm.cmd run build
npm.cmd audit
```

內容查核見 `docs/references.md`，學習設計見 `docs/learning-contract.md`，驗證紀錄見 `docs/test-report.md`。
